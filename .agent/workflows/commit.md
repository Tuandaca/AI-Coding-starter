2. Run `git diff` to see content of changes
3. Generate a conventional commit message based on changes (e.g. "feat: add login", "fix: resolve null pointer")
4. Show message to user
5. // turbo
   Run `git add .`
6. // turbo
   Run `git commit -m "MESSAGE"`
   (Replace MESSAGE with generated message)
